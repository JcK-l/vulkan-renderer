\doxysubsection{Detailed Description}
This file implements the Device class which is used to manage Vulkan devices. 

The Device class is part of the \doxylink{namespacevkf_1_1core}{vkf\+::core} namespace. It provides functionality to get queues and handle to the device. It also provides methods to check if a queue exists and to get the VMA allocator.
