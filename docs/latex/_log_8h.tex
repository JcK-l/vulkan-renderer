\doxysubsection{Detailed Description}
This file defines macros for logging messages at various levels of severity. 

The macros in this file use the spdlog library to log messages. The available macros are LOG\+\_\+\+INFO, LOG\+\_\+\+WARN, LOG\+\_\+\+ERROR, and LOG\+\_\+\+DEBUG. Each macro logs a message at a different level of severity. LOG\+\_\+\+ERROR also includes the filename and line number where the error occurred.
