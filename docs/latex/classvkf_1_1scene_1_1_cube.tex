\doxysubsection{Detailed Description}
This class manages \doxylink{classvkf_1_1scene_1_1_cube}{Cube} entities. 

This class provides an interface for creating and managing a \doxylink{classvkf_1_1scene_1_1_cube}{Cube} entity. A \doxylink{classvkf_1_1scene_1_1_cube}{Cube} entity in this context is a specific type of \doxylink{classvkf_1_1scene_1_1_prefab}{Prefab} that can be used in a 3D scene.
