This class manages Vulkan command pools.

It provides functionality to get the handle to the command pool and request command buffers.

Request command buffers.

This function requests command buffers from the command pool and stores them in an unordered map for later reference and automatic cleanup. see https://github.com/KhronosGroup/Vulkan-Hpp/blob/main/vk_raii_ProgrammingGuide.md section 4 for more info.

Parameters
levelThe level of the command buffers.
countThe number of command buffers.
Returns
A pair containing the key and a raw pointer to command buffers. Ownership is not transferred!
