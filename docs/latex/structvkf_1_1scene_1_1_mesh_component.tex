\doxysubsection{Detailed Description}
Struct for managing mesh data for an entity in a scene. 

This struct provides functionality to store and manage mesh data. It contains a shared pointer to a Buffer object to store the mesh data, a uint32\+\_\+t to store the number of vertices and a boolean to determine if the mesh should be drawn.
