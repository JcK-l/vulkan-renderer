Detailed Description

This file implements the Application class which is used for managing the main application.

The Application class is part of the vkf::platform namespace. It provides an interface for interacting with the main application, including running the main loop, initializing the logger, and creating the window, instance, surface, and device. It also provides methods for enabling instance extensions, instance layers, and device extensions.
