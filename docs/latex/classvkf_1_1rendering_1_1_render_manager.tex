\doxysubsection{Detailed Description}
This class manages Vulkan rendering operations. 

It provides an interface for interacting with Vulkan rendering operations, including beginning and ending frames and render passes, drawing, and managing the lifecycle of a \doxylink{classvkf_1_1rendering_1_1_renderer}{Renderer} object.
