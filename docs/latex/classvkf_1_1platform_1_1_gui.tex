\doxysubsection{Detailed Description}
This class manages Im\+Gui interfaces. 

It provides functionality to manage Im\+Gui interfaces, including creating images, image views, and handling Im\+Gui rendering. \doxylink{classvkf_1_1platform_1_1_gui}{Gui} is a subclass of Render\+Source and therefore provides methods for getting the image views and image count.
