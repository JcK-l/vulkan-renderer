\doxysubsection{Detailed Description}
This file declares the Event struct, which encapsulates various types of events. 

The Event struct is part of the vkf namespace. It provides a unified interface for handling different types of events, such as keyboard input, mouse movement, mouse button input, mouse scroll, window resize, and window close events.
