Detailed Description

This file implements the TransformComponent struct which is used for managing transformation data for an entity in a scene.

The TransformComponent struct is part of the vkf::scene namespace. It provides functionality to store and manage transformation data. It contains glm::vec3 members to store the position, rotation, and scale data.
