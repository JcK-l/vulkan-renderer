This file implements the Queue class which is used for managing Vulkan queues. 

The Queue class is part of the \doxylink{namespacevkf_1_1core}{vkf\+::core} namespace. It provides an interface for interacting with a Vulkan queue, including getting the handle to the queue, the family index, the queue properties, and the queue index. It also provides a method to check if the queue can present.
