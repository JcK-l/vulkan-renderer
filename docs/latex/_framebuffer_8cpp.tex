\doxysubsection{Detailed Description}
This file implements the Framebuffer class, which encapsulates a Vulkan framebuffer. 

The Framebuffer class is part of the \doxylink{namespacevkf_1_1core}{vkf\+::core} namespace. It provides an interface for creating and managing a Vulkan framebuffer. A framebuffer in Vulkan is a collection of specific attachments (images) that a render pass instance uses.
